# 포도 (PODO) - 안전 경로 안내 서비스

밤길 귀가가 불안한 노약자·여성·아동을 위한 안전 경로 안내 서비스입니다. 경찰서·소방서·편의점·CCTV 등 주변 공공시설을 기반으로 경로별 안전도를 분석하여, 다소 돌아가더라도 가장 안전한 길을 안내합니다.

## 프로젝트 정보

- 기간: 2025.04 ~ 2025.05 (3주)
- 인원: 4인 팀 프로젝트
- 원본 저장소: https://github.com/2025-SMHRD-IS-CLOUD-2/PODO

## 사용 기술

Java, Spring MVC, JSP, MyBatis, MySQL, T-Map API, jQuery

## 담당한 부분

- **로그인 화면**: 로그인 폼, 아이디·비밀번호 찾기, 회원가입 이동 처리
- **메인 지도 / 사이드바**: T-Map API 연동해서 지도 화면 구현. 경찰서·소방서·CCTV·편의점을 카테고리별 마커로 표시하고 토글로 선택 확인 가능하게 함
- **경로탐색 화면**: 출발지·목적지 입력하면 안전도 분석 결과 바탕으로 경로 제안, 경로정보+지도 레이아웃 구현
- **보호자 화면**: 보호자 등록/수정/삭제 리스트 화면 구현

## 트러블슈팅

**1. 프론트-백엔드 변수명 불일치로 데이터 매핑 안 되던 문제**
백엔드에서 넘겨주는 변수명이랑 프론트에서 쓰는 변수명이 서로 달라서 데이터가 제대로 안 붙는 문제가 있었음. 백엔드 팀원이랑 직접 얘기해서 변수명을 맞추기로 조율하고 해결함.

**2. T-Map 경로탐색 시 출발지 좌표 전달 문제**
처음엔 AJAX로 출발지 좌표를 다음 페이지(경로탐색 페이지)에 넘기려고 했는데, AJAX 응답은 페이지 이동하면 그대로 사라지는 값이라 페이지가 넘어가면 좌표가 유지되지 않는 문제가 있었음. 개발자도구(콘솔/네트워크 탭)로 확인해서 원인을 찾았고, URL 쿼리 파라미터 방식으로 바꿔서 해결함. 좌표를 `addressNavigation.jsp?startLat=...&startLon=...&endLat=...&endLon=...` 형태로 URL에 실어서 페이지를 이동시키고, 다음 페이지에서 그 값을 받아 T-Map 경로탐색 API 호출에 사용하도록 수정함.
