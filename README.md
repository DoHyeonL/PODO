# 포도 (PODO) - 안전 경로 안내 서비스

3인 팀 프로젝트입니다. 주변 안전 시설물(CCTV, 안전벨 등) 정보를 기반으로 경로를 점수화해서, 가장 빠른 길이 아니라 **가장 안전한 길**을 안내해주는 서비스입니다.

## 팀 정보

- 3인 팀 프로젝트
- 원본 저장소: https://github.com/2025-SMHRD-IS-CLOUD-2/PODO

## 사용 기술

- Java 17
- Servlet / JSP
- MyBatis (SQL 매핑)
- MySQL
- Maven (빌드 관리)

## 담당한 부분

- 로그인 화면
- 사이드바(메뉴) 화면
- 메인 화면 검색창
- 경로탐색(지도) 화면
- 보호자(함께하는 사람들) 화면

## 화면 구성

### 로그인
로그인 / 아이디·비밀번호 찾기 / 회원가입 이동 기능을 담당했습니다.

![로그인 화면](screenshots/login.png)

### 메인 + 사이드바
지도 위에 검색창과 사이드바 메뉴(안심 길찾기, 자주가는 장소, 함께하는 사람들, 알리미, 신고함)를 배치했습니다.

![사이드바 화면](screenshots/sidebar.png)

### 경로탐색
출발지·도착지를 검색하면 안전 경로와 예상 소요시간·거리를 지도 위에 표시합니다.

![경로탐색 화면](screenshots/route.png)

### 보호자 등록
보호자를 등록/수정/삭제할 수 있는 화면입니다.

![보호자 화면](screenshots/guardian.png)
